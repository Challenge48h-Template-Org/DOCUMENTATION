# DOCUMENTATION TEMPLATE CORPORATION

Ce repository doit contenir la totalité de votre documentation, que ce soit pour les utilisateurs finaux ou pour les développeurs. Vous pouvez organiser votre documentation en plusieurs fichiers Markdown, et les lier entre eux pour une navigation facile.

Vous êtes également autorisé à utiliser d'autres formats de documentations, (GitHub Wiki, GitBook, Notion, etc.) tant que vous fournissez un lien vers la documentation complète dans ce README.md.
